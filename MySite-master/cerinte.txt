HTML 
-fisiere separate pentru html si css.
-structurarea documentului folosind tagurile semantice: header, nav, section, article, aside, footer.
-folosirea elementelor h1-h6, p, img, ol, ul, a, table, div, span, iframe, meta, base.
-sa treaca testul de validare. 
 
CSS
-selectori: id, clasa, tag, elementContinut, elementCopil, al n-lea copil de tipul unui tag (:nth-of-type,:ntd-child)
-proprietati: width, height, color, background, text-align, font-size, border, padding, margin, display, visibility, vertical-align
-meniu drop-down, galerie de figuri cu figcaption
-layout: position(relative, absolute), z-index, float, flexbox, grid
-transformari, tranzitii, animatii
-responsive: %(width) , em (font-size), media-query (@media) 

JS 
- modificare dinamica de proprietati (ex: sursa unei imagini, url-ul unei ancore)
- modificare dinamica a stilului unui element (ex: culoare, dimeniuni, pozitie, font)
- modificare dinamica de eveniment (ex: el.onclick)
- selectare dupa id, tag, clasa, selectorCSS
- creare si stergere element (createElement si removeChild)
- inputuri functionale (buton,text,radio,checkbox)
- navigare in arbore (children, parentElement, ..)
- folosirea proprietatilor className, classList, getComputedStyle
- addEventListener cu param3=true
- obiectul event -proprietati (target, key, pageX/clientX)
                 -metode (stopPropagation, preventDefault)
- proprietate noua adaugata unui obiect (ex: numarul de click-uri care s-au dat pe un element)
- setTimeout, setInterval(cu 3 parametrii) , clearInterval
- window: open, close
- o metoda din clasele predefinite: Math, Array, String, Date	  
- innerHTML, localStorage
- folosirea ajax.
